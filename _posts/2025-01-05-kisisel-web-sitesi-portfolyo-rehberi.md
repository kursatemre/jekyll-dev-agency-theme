# Kişisel Web Sitesi ve Portfolyo Oluşturma Rehberi 2025

Dijital çağda kişisel markanız, kariyerinizin en değerli varlığıdır. Freelancer, sanatçı, tasarımcı, yazar, fotoğrafçı, developer veya herhangi bir profesyonel olun - etkileyici bir kişisel web sitesi ve portfolyo, sizi rakiplerinizden ayıran en güçlü silahınızdır.

## Kişisel Web Sitesi Neden Önemli?

### İstatistikler Konuşuyor

- İşverenlerin **70%'i** adayları online araştırıyor
- Freelancer'ların **93%'ü** portfolyo sitesi olanlar öncelikli değerlendiriliyor
- Profesyonel web sitesi olan kişiler **47% daha fazla** kazanıyor
- Müşterilerin **85%'i** çalışmadan önce online portfolyo istiyor

### Faydalar

1. **7/24 Dijital CV**: Sürekli çalışan size
2. **Profesyonellik**: Ciddi olduğunuzu gösterir
3. **Portfolio Showcase**: İşlerinizi sergileme alanı
4. **Lead Generation**: Müşteri kazanımı
5. **Personal Branding**: Marka inşası
6. **SEO**: Google'da bulunabilirlik
7. **Sosyal Kanıt**: Testimonial ve case study'ler
8. **Networking**: İş fırsatları

## Temel Sayfalar ve Bölümler

### 1. Ana Sayfa (Homepage)

**Hero Bölümü:**
- Profesyonel fotoğraf
- Güçlü headline: "Kim olduğunuz + Ne yaptığınız"
- Kısa value proposition (1-2 cümle)
- Primary CTA button

Örnek:
```
"Ben [İsim], [Şehir] bazlı [Meslek]
10+ yıl deneyimle, markaların dijital hikayelerini anlatmalarına yardımcı oluyorum."

[Portfolyomu İncele] [Benimle Çalış]
```

**Featured Work:**
- En iyi 3-6 proje
- Thumbnail'ler
- Proje başlığı
- Kısa açıklama
- Case study linki

**Social Proof:**
- Client logoları
- Testimonial'lar (2-3 tane)
- Rakamlarla başarılar
- Medya mentions

### 2. Hakkımda (About)

**Hikayeniz:**
- Kişisel yolculuk
- Uzmanlık alanları
- Çalışma felsefeniz
- Değerleriniz

**Profesyonel Bilgiler:**
- Eğitim
- Deneyim
- Beceriler (skill bars)
- Sertifikalar
- Ödüller

**Kişisel Touch:**
- Hobilkr
- İlham kaynakları
- Fun facts
- Lifestyle fotoğraflar

**İstatistikler:**
```
[250+] Tamamlanan Proje
[50+] Mutlu Müşteri
[10+] Yıllık Deneyim
[15+] Ödül & Sertifika
```

### 3. Portfolyo/Çalışmalarım

**Filtreleme:**
- Kategori bazlı
- Sektör bazlı
- Hizmet türü
- Yıl

**Proje Detay Sayfası:**

**Başlık ve Özet:**
- Proje adı
- Client/Brand
- Proje türü
- Tarih

**Challenge (Problem):**
- Müşteri sorunu
- Proje hedefleri

**Solution (Çözüm):**
- Yaklaşımınız
- Süreç
- Kullanılan araçlar

**Visuals:**
- High-quality images
- Mockup'lar
- Before/After

**Results (Sonuçlar):**
- Ölçülebilir metrikler
- Client feedback

**CTA:**

### 4. Hizmetler/Services

**Hizmet Paketleri:**

**Bronze/Basic:**
- Kapsamı
- Deliverable'lar
- Timeline
- Fiyat (opsiyonel)

**Silver/Standard:**
- Daha kapsamlı
- Ek özellikler
- Most popular badge

**Gold/Premium:**
- Full-service
- Premium support
- Fastest delivery

**Çalışma Süreci:**
```
1. Keşif Görüşmesi (30 dk ücretsiz)
2. Teklif & Sözleşme (24 saat)
3. Ön Ödeme & Başlangıç (1 gün)
4. Tasarım/Geliştirme (1-4 hafta)
5. Revizyon (2 tur dahil)
6. Teslimat & Eğitim
7. Destek (30 gün ücretsiz)
```

**FAQ:**
- Sık sorulan 8-10 soru
- Net ve kısa yanıtlar
- Concern'leri address edin

### 5. Blog

**Neden Blog?**
- SEO için kritik

- How-to guides

**Kolay Erişim:**
- Basit form (3-5 alan)

**Form Alanları:**

**Response Time:**
"24 saat içinde geri dönüş garantisi"

**Konum:**
- Şehir (tam adres gerekli değil)
- Çalışma zamanları
- Time zone

## Tasarım Prensipleri

### Minimalist ve Temiz

**Neden?**
- İçeriğiniz öne çıkar
- Professional görünüm
- Hızlı yüklenme
- Uzun ömürlü

**Temel Öğeler:**
- Bol beyaz alan
- Sınırlı renk paleti (2-3 renk)
- Tutarlı tipografi (2 font max)
- Grid-based layout
- Subtle animations

### Typography (Tipografi)

**Font Seçimi:**
- **Sans-serif** (modern, temiz):
  - Inter, Poppins, Montserrat, Roboto
- **Serif** (klasik, elegant):
  - Playfair Display, Merriweather, Lora

**Hiyerarşi:**
```css
H1: 48-72px (Hero başlıklar)
H2: 36-48px (Bölüm başlıkları)
H3: 24-30px (Alt başlıklar)
Body: 16-18px (İçerik)
Small: 14px (Caption, footer)
```

**Line Height:**
- Başlıklar: 1.2-1.3
- Body text: 1.5-1.7

### Renk Paleti

**Brand Colors:**
- Primary: Markanızı temsil eden
- Secondary: Tamamlayıcı
- Accent: CTA button'lar için
- Neutrals: Gri tonları

**Örnekler:**

**Minimal & Professional:**
```
Primary: #2C3E50 (Koyu Mavi-Gri)
Secondary: #ECF0F1 (Açık Gri)
Accent: #3498DB (Mavi)
Background: #FFFFFF
Text: #2C3E50
```

**Creative & Bold:**
```
Primary: #FF6B6B (Mercan)
Secondary: #4ECDC4 (Turkuaz)
Accent: #FFE66D (Sarı)
Background: #F7F7F7
Text: #292F36
```

**Dark & Modern:**
```
Primary: #1A1A2E (Koyu Lacivert)
Secondary: #16213E (Lacivert)
Accent: #E94560 (Kırmızı)
Background: #0F3460
Text: #EAEAEA
```

### Responsive Design

**Mobile-First Approach:**
- %60+ traffic mobilde
- Touch-friendly butonlar (min 44x44px)
- Kolay navigasyon
- Hızlı yükleme

**Breakpoints:**
```css
Mobile: < 768px
Tablet: 768px - 1024px
Desktop: 1024px - 1440px
Large: > 1440px
```

**WordPress:**
**Avantajlar:**

- 50,000+ tema

- Blog entegre

**Webflow:**
**Avantajlar:**

**Maliyet:** $14-42/ay + tasarım

**Squarespace:**
**Avantajlar:**
- Çok kolay

- All-in-one
**Dezavantajlar:**
- Sınırlı özelleştirme
**Maliyet:** $16-49/ay

**Custom (React/Next.js):**
**Avantajlar:**
- Tam kontrol
- Benzersiz
- Hızlı
**Maliyet:** ₺15K-40K

### Essential Features

**Must-Have:**
- SSL sertifikası (HTTPS)
- Responsive design
- Fast loading (2 sn altı)
- Contact form
- Social media links
- Google Analytics

**Nice-to-Have:**
- Live chat

**Speed Matters:**
- Görsel optimizasyonu (WebP format)
- Lazy loading
- CDN kullanımı
- Minify CSS/JS
- Cache mekanizması

**Hedef:**
- Google PageSpeed: 90+
- Yüklenme: < 2 saniye
- Time to Interactive: < 3.5 sn

### SEO Basics

**On-Page SEO:**
- Title tags optimize
- Meta descriptions
- H1-H6 hiyerarşi
- Image alt texts
- Internal linking
- URL structure

**Technical SEO:**
- XML sitemap
- Robots.txt
- Schema markup
- Canonical URLs
- Mobile-friendly

**Content SEO:**
- Anahtar kelime araştırması
- İsim + meslek + şehir
- "freelance [meslek] [şehir]"
- Düzenli blog

## Portfolyo Showcase Best Practices

### Proje Seçimi

**Kalite > Quantity:**
- 6-12 en iyi proje yeter
- Çeşitlilik gösterin
- Recent work öncelik
- Eski projeleri temizleyin

**Çeşitlilik:**
- Farklı sektörler
- Farklı proje türleri
- Farklı stil ve yaklaşımlar
- B2B ve B2C mix

**Storytelling:**
Her proje bir hikaye:
- Problem
- Süreç
- Çözüm
- Sonuç

### Görsel Sunum

**Photography:**
- Profesyonel fotoğraf çekimi
- Tutarlı lighting
- High resolution (min 1920px genişlik)
- Neutral backgrounds

**Mockups:**
- Laptop/telefon mockup'ları
- Contextual mockups
- Multiple angles
- Realistic presentations

**Before/After:**
- Dönüşümü göster
- Slider kullan
- Impact'i vurgula

### Case Study Yapısı

```markdown
# [Proje Adı] - [Client]

## Overview
- Client: [İsim]
- Industry: [Sektör]
- Services: [Hizmetler]
- Duration: [Süre]
- Tools: [Araçlar]

## The Challenge
[Problem detayı - 2-3 paragraf]

## The Solution
[Yaklaşım ve süreç - 3-4 paragraf]

[Process Görselleri]

## Results
- [Metrik 1]: X% increase
- [Metrik 2]: Y+ new customers

## Client Testimonial

[İletişime Geç]
```

## Başarı Metrikleri

### Tracking

**Google Analytics:**

- Bounce rate
- Average session duration
- Traffic sources

**Goals:**
- Contact form submissions
- Email signups
- Portfolio clicks
- Social media clicks

### KPIs

**Traffic:**
- Hedef: 500-2.000/ay (ilk 6 ay)
- Organic: %40+
- Direct: %30
- Social: %20
- Referral: %10

**Engagement:**
- Bounce rate: %40-60
- Pages/session: 2.5-4
- Session duration: 2-4 dakika

**Conversions:**
- Contact rate: %2-5
- Response rate: %80+
- Conversion to client: %20-40
